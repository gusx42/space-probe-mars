# Mars Exploration

Create a mars space probe exploration

## How use this

Initial configuration:
* Area size: 5x5
* Initial position: (0,0,N) = x = 0, y = 0 and camera = North

Moves:
* M = move bot to foward
* L = rote the bot to the left
* R = rote the bot to he rigth

Move robot to positon:
* `MMRMMRMM` = (2,0,S) 
* `MML` = (0,2,W)  
* `MMRMMLMMRMM` = (4,4,E)
* `AAAA` = Error 
* `MMMMM` = Error
* `MLX` = Error
